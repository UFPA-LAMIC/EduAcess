# EduAcess
O EduAccess Pro (Viva! Universidade 2025) utiliza Inteligência Artificial local e algoritmos para adaptar materiais didáticos universitários. A ferramenta converte PDFs em formatos acessíveis (áudios narrados e textos otimizados) para discentes com deficiência visual, TDAH, autismo e dislexia, promovendo autonomia e inclusão no ensino superior.
